# codementor-ai
CodeMentor AI là nền tảng học lập trình tích hợp AI giúp người mới bắt đầu trở thành developer thực thụ thông qua mentor AI thông minh, roadmap cá nhân hóa, coding trực tiếp, sửa lỗi thời gian thực, project thực tế và hệ thống học tập thích ứng hiện đại như VSCode + Cursor + Duolingo trong một nền tảng duy nhất.
